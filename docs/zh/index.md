# 欢迎来到 AmritaSense

AmritaSense 是一个**通用工作流编排引擎**。它将工作流视为可编程的执行流，用“指令”代替“图”，让你用最直接的方式描述任务逻辑。

> **Any is thing all you need.**

在这里，任何异步任务都可以拆解为节点与指令的组合。无论是构建 AI Agent、自动化流程，还是编排复杂业务逻辑，AmritaSense 都能提供强大而灵活的支持。
